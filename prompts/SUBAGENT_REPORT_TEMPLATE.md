# Subagent Report Template

## Summary

What was completed in 3-7 bullets.

## Files changed

List each changed file and why it changed.

## Tests run

Commands run and results.

## Benchmark impact

State one of:

- not measured
- no expected impact
- measured; include numbers

## Decisions needed

List any decisions the strategic agent/user must make.

## Risks

List implementation, quality, performance, or architecture risks.

## Next recommended task

State the next smallest useful task.
