# QUALITY_RUBRIC.md — PhASCII Visual Evaluation

Use this rubric when comparing generated ASCII outputs.

## Scale

| Score | Meaning |
|---:|---|
| 0 | Not recognizable. |
| 1 | Weak relationship to original. |
| 2 | Recognizable only with original beside it. |
| 3 | Clearly recognizable. |
| 4 | Strongly recognizable and visually pleasing. |
| 5 | Excellent ASCII representation. |

## Categories

Score each output in these categories:

- silhouette
- contrast
- subject recognizability
- noise level
- texture/detail usefulness
- text/logo readability, if relevant
- overall preference

## Required comparison format

```text
Input image:
Mode A:
Mode B:
Preferred mode:
Reason:
Scores:
- silhouette:
- contrast:
- recognizability:
- noise:
- overall:
Decision:
```
